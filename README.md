# Simulación de Microempresa de Servicios Eléctricos

Este repositorio contiene el código fuente y la documentación para el desarrollo de una simulación orientada a la creación de una microempresa de servicios eléctricos. 

## 🎯 Objetivo del Proyecto
El objetivo principal es desarrollar una simulación de la creación de una microempresa de servicios eléctricos, con el fin de analizar y optimizar sus procesos administrativos y técnicos. A través de este sistema, se busca mitigar problemas comunes como la ineficiencia operativa, la mala planificación de actividades y el descontrol en los costos.
